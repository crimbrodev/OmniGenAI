# OmniGen Multiverse

Мультимодальная AI-платформа с интеграцией Gemini API.

## Деплой

Проект автоматически деплоится на GitHub Pages при пуше в main ветку.

## Локальная разработка

```bash
npm install
npm run dev
```

## Настройка API ключа

1. Получите API ключ в [Google AI Studio](https://aistudio.google.com/)
2. Нажмите на иконку ключа в sidebar
3. Введите ваш API ключ

## Технологии

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Google Gemini API"# OmniGenAI"
